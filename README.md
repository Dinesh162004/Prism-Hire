## 🧠 What is Prism Hire?

**Prism Hire** is a modern, AI-driven platform that serves as an all-in-one academic and career companion for students. It combines intelligent study tools, personalized mentorship, and realistic interview preparation — all powered by **Google Gemini 1.5 Flash**.

Unlike generic prep tools, Prism Hire uses **isolated sessions** to keep your study mindset and placement mindset completely separate — helping you stay focused, organized, and confident at every stage.

---

## ✨ Core Features

### 📚 Study Session

- **AI Doubt Solver:** Ask any academic question and get instant, detailed AI-powered explanations.
- **PDF Analyzer:** Upload any study material — the AI reads, parses, and answers questions from it.
- **Study Planner:** Organize your topics, deadlines, and learning roadmap in one place.
- **Mentorship Module:** Connect with mentors for guided academic support and career advice.

### 💼 Placement Session

- **AI Mock Interview:** Dynamic, role-specific MCQ assessments generated by Gemini AI.
- **Face-Time AI Interview:** Real-time voice interview simulation with live transcript and AI feedback.
- **Resume ATS Analyzer:** Upload resume + JD — get ATS match score and improvement tips.
- **Saved Questions Library:** Bookmark important questions, organized by job role.
- **Mini Quiz Mode:** Retake quizzes from your saved library to reinforce weak areas.

### 🔐 Platform Features

- **Isolated Sessions:** Study and Placement sessions are completely independent, no context bleed.
- **JWT Authentication:** Secure login, registration, and session persistence.
- **Profile Management:** Custom profile picture, name, and auto-sync across the app.
- **Progress Tracking:** Scores, streaks, and assessment history at a glance.

## 🚀 Local Development Setup

### Prerequisites

- Java 17+ and Maven
- Node.js v18+
- MongoDB local instance or Atlas URI
- Google Gemini API Key

### 1. Backend Setup (Spring Boot)

Open the `server-java` folder in your IDE (IntelliJ, Eclipse, or VS Code).
Update the `application.properties` or set environment variables:

```properties
spring.data.mongodb.uri=your_mongodb_atlas_connection_string
jwt.secret=your_super_secret_jwt_key
gemini.api.key=your_gemini_api_key
server.port=8080
```

Run the application using Maven or your IDE's run configuration:

```bash
cd server-java
mvn spring-boot:run
```

### 2. Frontend Setup (React/Vite)

Navigate to the `client` directory:

```bash
cd client
npm install
```

Configure your `.env` file for the Vite app:

```env
VITE_API_URL=http://localhost:8080
```

Start the frontend development server:

```bash
npm run dev
```

> App will be running at `http://localhost:5173`
